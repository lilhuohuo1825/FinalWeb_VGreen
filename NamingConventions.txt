Quy tắc chung
1. Tất cả tên file và thư mục phải được viết bằng chữ thường (lowercase), không dấu, không chứa khoảng trắng.
2. Đẳt tên file theo ngôn ngữ là tiếng anh (có gợi ý một số tên rồi á)
3. Các từ được phân tách bằng gạch dưới (_)
4. Không sử dụng ký tự đặc biệt như (! @ # $ % ^ & * ( ) [ ] { }...)
5. Tên file phải thể hiện rõ ràng chức năng hoặc nội dung chính của tệp.
6. Độ dài tên file nên ngắn gọn, tránh mô tả dài dòng.
7. Mỗi file bắt buộc phải có phần mở rộng (extension) tương ứng, ví dụ: .html, .css, .js, .json, .csv. (bỏ vào thư mục tương ứng trong cấu trúc file luôn nha)

=> t có gợi ý đặt tên một số file trong cây á, tham khảo nhe
